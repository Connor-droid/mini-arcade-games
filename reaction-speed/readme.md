TITLE : Reaction Speed


You test how fast your reaction is:

- Click **Start**
- Wait for the button to turn **green**
- Tap as quickly as you can
- See your **last time**, **best time**, and **attempt count**

 Game Details

- Tech: HTML, CSS, JavaScript 
- Folder: 'reaction-speed'
- Files:
 - 'index.html' – structure and layout
 - 'style.css' – styling
 - 'script.js' – game logic
 - 'readme.md' - description of the mini game

Running process

  Clone this repository:

   git clone https://github.com/Connor-droid/mini-arcade-games.git
   cd reaction-speed
